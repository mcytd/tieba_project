# Tieba Project / 贴吧项目

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)](https://www.python.org/)
[![Django Version](https://img.shields.io/badge/django-4.0%2B-green)](https://www.djangoproject.com/)

A Django-based forum / bulletin board system (like a "Tieba") with bilingual support.  
一个基于 Django 的论坛/贴吧系统，支持中英双语。

---

## 🚀 Features / 功能特点

- **User authentication** (register, login, logout) / 用户认证（注册、登录、登出）
- **Post threads and replies** / 发布主题帖和回复
- **Category / board management** / 版块分类管理
- **Search and pagination** / 搜索与分页
- **Responsive UI** (Bootstrap) / 响应式界面（Bootstrap）
- **i18n ready** (English/Chinese) / 国际化支持（中英文切换）

---

## 📦 Tech Stack / 技术栈

- Backend: Django 4.x + SQLite (or PostgreSQL)  
- Frontend: Bootstrap 5 + jQuery  
- Others: Django REST Framework (optional)

---
